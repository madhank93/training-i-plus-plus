### Time to write first Go program (Hello-World)

Create file ```hello.go``` with content:

```go
package main
import "fmt"

func main(){
    fmt.Printf("Hello World\n")
}
```
Import built-in package ```fmt``` and used it’s function ```Printf``` to print “Hello World” to console.
